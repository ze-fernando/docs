# Morfologia
É o estudo da palavra
- Classe
- Estrutura
- Formação

## Classes de palavras
Existem dois tipos de classe de palavras: *variáveis* e *invariáveis*
Variáveis são palavras que variam em:
- Gênero
	- Lind**o** | Lind**a**
- Número
	- Feli**z** | Feli**zes**



| Variáveis   | Invariáveis | Ambos    |
| ----------- | ----------- | -------- |
| Substantivo | *Advérbio*  |          |
| *Adjetivo*  | Preposição  | Pronomes |
| Artigos     | Interjeição | Numerais |
| Verbo       | Conjução    |          |

## 1. Substantivo
Substantivo é a classe que da nome ao seres em geral. Portanto, é qualquer palavra marcada (ou que marcada poderia estar) por um **artigo**, **pronome**, **adjetivo** ou **numeral**.
Exemplos:
- Pela **manhã**, a **bola** do **vizinho** está sempre no **meio** do meu **pátio**.
> [!WARNING] Atenção
> A frase de exemplo depende do *contexto* 
### 1.1 Substantivo Concreto
São substantivos que *existem* por si só, ou seja, tem vida própria. Não dependem de outro ser para existir.
Exemplos:
- Livro
- Lápis
- Fada
- Deus 
### 1.2 Substantivo Abstrato 
São substantivos que dependem de outros para existirem, geralmente designam:

| Ação    | Qualidade | Sensação | Sentimento |
| ------- | --------- | -------- | ---------- |
| Pesca   | Beleza    | Dor      | Medo       |
| Traição | Pureza    | Frio     | Saudade    |
| Amor    | Rapidez   | Calor    | Ciúme      |

## 2. Adjetivo
Adjetivo é a classe que da características e/ou refere-se ao substantivo/pronome.
Exemplos:
- A **jovem** secretária chegou
- A jovem **discreta** chegou

### 2.1 Grau Comparativo
- Igualdade
	- Tão | Como | Tanto | Quanto
		- Luís é tão calmo quanto Felipe
- Inferioridade
	- Menos | Do que
		- Luís é menos calmo que Felipe
- Superioridade
	- Mais | Do que
		- Luís é mais calmo que Felipe

### 2.2 Grau Superlativo
- Relativo
	- Inferioridade (comparação de um ser com um todo)
		- Luís é o menos calmo da família
	- Superioridade
		- Paulo é o mais tímido do time
- Absoluto
	- Analítico
		- Maria é **muito** linda
	- Sintético
		- Maria é lind**íssima**
### 3. Advérbio
Advérbio é a classe que expressa circunstância e/ou refere-se ao verbo, adjetivo ou advérbio.
Exemplos:
- **Ontem** ele chegou **cedo**
	- Circunstância de tempo
- Ele chegou **mais** **cedo**
	- Circunstância de tempo
	- Ideia de intensidade
- Ela chegou **muito** linda
	- Circunstância de modo

### 4. Preposições
Preposição é a classe responsável por ligar dois termos de uma oração, sendo elas:
> [!WARNING] Atenção
> Preposições principais estão ==marcadas==

| A     | C       | D      | E      | P        | S     |
| ----- | ------- | ------ | ------ | -------- | ----- |
| ==A== | ==Com== | ==De== | ==Em== | ==Para== | Sob   |
| Ante  | Contra  | Desde  | Entre  | Perante  | Sobre |
| Até   |         |        |        | ==Por==  | Sem   |
| Após  |         |        |        |          |       |
### 5. Artigos
Artigo é a classe que acompanha o substantivo, sendo eles:
- Definidos
	- O | Os | A | As
- Indefinidos
	- Um | Uns | Uma | Umas
