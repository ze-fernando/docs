# Acentuação Gráfica

### 1. Acentos e Sinais
#### 1.1 Acentos marcam a sílaba tônica de uma palavra

| Simbólo | Nome          | Exemplo |
| ------- | ------------- | ------- |
| `       | Agudo         | Lépido  |
| ^       | Circunflexo   | Bônus   |
| ´       | Grave (crase) | a+a = á |

#### 1.2 Cada sinal tem uma função diferente

| Sinal | Nome      | Função                      |
| ----- | --------- | --------------------------- |
| ~     | Til       | Marca a nasalização         |
| '     | Apóstrofo | Marca a supressão de letras |
| ¨     | Trema     | **Marcava** o som do "u"    |

### 2. Sílaba Tônica
Sílaba tônica é a silaba mais forte "foneticamente" falando, e ela se define se a palavra é
- Oxítona
	- Sílaba tônica é a **última** sílaba
- Paroxítona
	- Sílaba tônica é a **penúltima** sílaba
- Proparoxítona
	- Sílaba tônica é a **antepenúltima** sílaba

### 3. Regras de Acentuação
#### 3.1 Proparoxítonas
> Todas as proparoxítonas são acentuada

#### 3.2 Oxítonas
São acentuadas as oxítonas terminadas em:
- A(s) | E(s) | O(s) | Em | Ens 
- Sofás | Vendê-lo | Alguém
#### 3.3 Monossílabos Tônicos 
São acentuados os monossílabos tônicos terminados em:
- A(s) | E(s) | O(s)
- Pó | Fé | Chá
#### 3.3 Paroxítonas
São acentuadas as paroxítonas **não** terminadas em:
- A(s) | E(s) | O(s) | Em | Ens 
- Táxi | Vírus | Lápis
### 4. Regras adicionais
#### 4.1 Hiatos "i" e "u"
São acentuados os hiatos "i" e "u" quando:
- Formarem sílabas sós ou com "s"
	- País
- São precedidos de vogal
	- Baú
- Não são seguidos de "nh"
	- Faísca
### 5. Ditongos Tônicos abertos
São acentuados os ditongos tônicos abertos:
- Éu | Ói | Éi 
	- Em posição de oxíntona
- Herói | Chapéu | Papéis
