# Regras gerais

## 1. Adição e Subtração (Regras gerais)

### 1.1 Comutatividade
- É uma operação **comutativa** se, para quaisquer dois elementos $a$ e $b$, a ordem dos operandos não alterar o resultado
- $a + b = b + a$

### l.2  Associatividade
- É uma operação **associativa** se, para quaisquer três elementos $a$, $b$ e $c$, o modo de agrupar não altera o resultado
- $a + (b + c) = (a + b) + c$

### 1.3 Existência do elemento neutro
- Possui um elemento que não altera nada na operação, independente da sua posição
- $a + 0 = a$

### 1.4 Existência do elemento oposto
- Possui um elemento que é o exato oposto do atual, zerando a equação quando utilizado
- $a + (-a) = 0$


## 2. Multiplicação e "Divisão" (Regras gerais)
> Somente se aplica a divisão se pensarmos nela como inverso da multiplicação

### 2.1 Comutatividade
- É uma operação **comutativa** se, para quaisquer dois elementos $a$ e $b$, a ordem dos operandos não alterar o resultado
- $a \cdot b = b \cdot a$

### 2.2 Associatividade
- É uma operação **associativa** se, para quaisquer três elementos $a$, $b$ e $c$, o modo de agrupar não altera o resultado
- $a \cdot (b \cdot c) = (a \cdot b) \cdot c$

### 2.3 Existência do elemento neutro
- Possui um elemento que não altera nada na operação, independente da sua posição
- $a \cdot 1 = a$

### 2.4 Existência do elemento oposto
- Possui um elemento que é o exato oposto do atual, zerando a equação quando utilizado
- $a + (a^{-1}) = 0$

### 2.5 Distributividade
> Fatoração é o inverso da distributividade

- É uma operação **distributiva** se, para quaisquer elementos $a$, $b$ e $c$, uma operação pode ser distribuída sobre outra sem alterar o resultado
- $a \cdot (b + c) = a \cdot b + a \cdot c$


## Fatoração (Casos de uso)

### Caso 1
- $a \cdot b + a \cdot c = a \cdot (b + c)$

### Caso 2
- $(a + b)^2 = a^2 + 2ab + b^2$

### Caso 3
- $a^2 - b^2 = (a + b) \cdot (b - a)$

### Caso 4
- $(a + b)^3 = a^3 + 3a^{2}b + 3ab^2 + b^3$


### Caso 5
- $(a + b)^3 \neq a^3 + b^3$
- $a^3 + b^3 = (a + b) \cdot (a^2 - ab + b^2)$
